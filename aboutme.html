I also started working on an IoT based project called Smart Traffic Lights in which by the help of sensors can release the traffic for the road that has more people. After releasing the traffic, it will once again check for the road that has most traffic (in case of a crossroads) and then release that traffic and so on. Based on this data other users can even see the statistics by installing an app and can start their journey according to the real-time data shown.
