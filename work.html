My Work

So in the past two months I was working on a technology called IoT (Internet of Things). So first I started with understanding protocols and learned in very detail about the various protocols required. I learned about the OSI model and learned about various protocols starting from Application Layer to Physical Layer. I have done a lot of research on how protocols work and even wrote a blog on it. I have completed over 3 courses on Coursera related to IoT and watched over 44 videos to be exact on Arduino Programming. Though I have worked a lot but I still feel inside that I want to work on my topic of interest i.e Web Development. Next section will display a project I have done on it. I also started working on an IoT based project called Smart Traffic Lights in which by the help of sensors can release the traffic for the road that has more people. After releasing the traffic, it will once again check for the road that has most traffic (in case of a crossroads) and then release that traffic and so on. Based on this data other users can even see the statistics by installing an app and can start their journey according to the real-time data shown. Unfortunately I am not working on this anymore since I want to work on web-based applications and explore Django more.
